# Rocket Mouse Game

Based on the book **_Infinite Runner in Phaser 3 with TypeScript_**.

[Link](https://rocket-mouse-game.vercel.app/ 'Click to enter')

## Main dependencies:

- [Phaser](https://phaser.io/ 'Phaser')
- [Vite](https://v2.vitejs.dev/ 'Vite')
- [TypeScript](https://www.typescriptlang.org/ 'TypeScript')
- [ESLint](https://eslint.org/ 'ESLint')
- [Prettier](https://prettier.io/ 'Prettier')

## Controls:

- **space/touch the screen**: start/move the mouse/restart

## Features:

- Progressive Web App
